# CalculationCore

## A calculation application that performs computations using built-in mathematical formulas.

### Programming Language Used: C++

### Features

### Performs calculations using predefined mathematical formulas

### . Clean and modular code structure
### .Easy to extend with additional formulas
### .Fast and lightweight

## Installation / Build

Ensure you have a C++ compiler installed (e.g., g++, Clang, or MSVC)

Clone the repository:

git clone https://github.com/yourusername/CalculationCore.git


Navigate into the project folder:

cd CalculationCore


Compile the project:

g++ -o CalculationCore main.cpp


(Adjust the compile command depending on your file structure.)

Usage

Run the compiled application:

./CalculationCore


Use the interface to select a formula or enter values required for the calculation.
